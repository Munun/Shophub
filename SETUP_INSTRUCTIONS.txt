STEP 1: EXTRACT FILES
----------------------
1. Extract the downloaded file to your Desktop
2. You should see two folders: backend/ and frontend/

STEP 2: BACKEND SETUP (5 minutes)
----------------------------------
1. Open Terminal and navigate:
   cd ~/Desktop/complete-ecommerce/backend

2. Install dependencies:
   npm install

3. Create database:
   createdb shophub
   psql shophub < schema.sql

4. Configure environment:
   cp .env.example .env
   nano .env
   
   Update these:
   - DB_USER=your_mac_username (replace with YOUR username)
   - STRIPE_SECRET_KEY=sk_test_... (get from stripe.com/test/apikeys)
   - STRIPE_PUBLISHABLE_KEY=pk_test_... (get from stripe.com/test/apikeys)

5. Start backend:
   npm run dev
   
   Should see: âœ… Server running on http://localhost:5001

STEP 3: FRONTEND SETUP (3 minutes)
-----------------------------------
1. Open NEW Terminal window and navigate:
   cd ~/Desktop/complete-ecommerce/frontend

2. Install dependencies:
   npm install

3. Start frontend:
   npm run dev
   
   Should see: Local: http://localhost:5173

STEP 4: TEST THE APP
--------------------
1. Open browser: http://localhost:5173
2. Click "Sign Up" and create an account
3. Browse products and add to cart
4. Go to checkout
5. Use test card: 4242 4242 4242 4242
